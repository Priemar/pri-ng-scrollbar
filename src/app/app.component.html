pri-ng-scrollbar

Angular 7 custom scrollbar, with native scrolling mechanism, outside of the angular change detection for better performance. Fallback for devices with floating.

Hope nobody feels offended by such an unstyled ugly page... if there is time left i'll add some styles i promise <<.

Default:

overflowX: auto
overflowY: auto

Mix: native and custom scrollbar

overflowX: scroll
overflowY: native-scroll

Custom margins

marginX => horizontal scrollbar => "10 20" => like in css: top/bottom left/right [top margin only if xPosition = 'top']
marginY: vertical scrollbar => "10 20 40 40" => - " - : top right left bottom [left margin only if yPosition = 'left']

Custom margins

xPosition: top => horizontal scrollbar
overflowY: left => vertical scrollbar
